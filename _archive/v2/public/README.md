This folder contains the current Shine and Speak brand/reference assets.

Preferred reusable logo assets:

- `shine-and-speak-logo-horizontal-transparent.png`
  Use for headers, nav bars, hero sections, and most website placements.
- `shine-and-speak-logo-stacked-transparent.png`
  Use where a taller lockup fits better, such as cards, social graphics, and centered sections.
- `shine-and-speak-logo-mark-transparent.png`
  Use as the icon/brand mark for favicons, badges, small UI elements, or where text is not needed.
- `shine-and-speak-favicon-mark-transparent.png`
  Square favicon/app-icon version of the logo mark with padding for small-size legibility.
- `shine-and-speak-favicon.ico`
  Browser favicon file generated from the favicon mark.

Reference-only assets:

- `shine-teoh-headshot-thumbnail.webp`
  Small portrait thumbnail reference.
- `shine-teoh-welcome-banner-reference.jpg`
  Legacy welcome/banner reference image, not a transparent logo.

Notes for future build agents:

- Prefer the transparent PNG logo files above for any new site or UI work.
- Use the horizontal logo by default unless the layout clearly needs the stacked or mark-only version.
- Use the favicon PNG or ICO specifically for browser tab icons and app icon slots.
- Do not recreate the logo from scratch while these files exist.
