Shared image library for all website versions.

Start with `../ASSET_MANIFEST.md`.

That manifest is the source of truth for:
- canonical shared assets
- temporary placeholders
- missing assets still to source
- guidance for build agents

Folder intent:
- `brand/` - canonical logo and favicon assets
- `shine/` - real client-provided Shine imagery
- `communication/` - business communication assets such as QR codes
- `placeholders/` - temporary stock stand-ins
- `og/` - temporary social share assets
- `badges/` - temporary trust badge placeholders
